# Roadmap

## Phase 1: 可用雷达

- Vite/React 静态仪表盘。
- RSS/HTML 自动更新。
- GitHub Actions + Pages 部署。
- 来源健康、筛选、家庭友好评分。

## Phase 2: 更强的家庭计划

- Center Parcs、Tropical Islands、Therme Erding、Familotel 等家庭度假源。
- Eventbrite/Fever/Groupon/Rausgegangen 的活动折扣适配器。
- 柏林出发机场偏好：BER 优先，也可纳入 Hamburg/Leipzig/Dresden 作为备选。
- 婴儿友好细分：婴儿床、电梯、厨房、短途转机、泳池、儿童餐、 stroller-friendly。

## Phase 3: 个性化与提醒

- 收藏、排除、不感兴趣、已看过。
- Telegram/Email/Slack 每周摘要。
- 指定目的地监控：例如土耳其 Side 某酒店、埃及 Hurghada 全包、MSC 地中海邮轮。
- 预算阈值和出行窗口提醒。

## Phase 4: 半自动行程

- 将优惠条目转为候选行程。
- 按宝宝作息、老人同行、交通时间、天气、签证/证件要求做行程评分。
- 导出到日历或 Notion/Google Sheet。
