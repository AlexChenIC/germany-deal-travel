# 重点折扣设计审查

更新时间：2026-07-01

## 当前判断

网站当前的信息源覆盖已经足够广，但默认体验容易变成“旅行信息流”。对家庭真实决策来说，更重要的是先看到少量高价值信号：

- 明确 40%+ 或 50%+ 的大幅折扣。
- 官方 Last Minute、全包/机酒套餐、酒店会员价、人工精选榜单。
- 柏林出发、德国短住、泳池/SPA/温泉、儿童或宝宝友好。
- 能直接进入核验流程的链接，而不是泛泛灵感。

因此新增 `重点折扣` 页面，并把它设为默认首页。普通活动和普通灵感仍保留在 `推荐雷达`，但不再抢占第一屏。

## 页面职责

`重点折扣` 页面只做两件事：

1. 展示最值得长期监控的折扣来源，以及每个来源的使用场景和风险。
2. 从当前抓取结果中筛出折扣强、家庭适配高、旅行本体相关的条目。

## 降权规则

以下内容即使出现在旅行分类，也不应进入首页重点区：

- 信用卡、返现、PayPal、礼品卡、机场免税购物。
- Uber/交通券等间接优惠，除非与明确旅行套餐绑定。
- Adults-only 或疑似不适合带婴儿的酒店。
- 只有泛泛灵感、没有价格或折扣信号的文章。

## 下一步

- 给 TUI、Travelzoo、Travelcircus、Secret Escapes 增加更稳定的专用抓取器或浏览器检查。
- 对大幅折扣条目补“最终可订价格/家庭人数/婴儿政策/空调和泳池证据”二次核验。
- 在每日自动化摘要中加入 `重点折扣新增` 计数。
