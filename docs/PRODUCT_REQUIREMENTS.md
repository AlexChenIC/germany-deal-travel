# 柏林家庭出行雷达：一期需求整理

## 目标

为居住在柏林的家庭建立一个可持续更新的信息雷达，聚合柏林市内活动、柏林周边短途、德国/欧洲度假、从德国出发的阳光目的地、全包酒店套餐和邮轮折扣。使用者画像按两位成人、宝宝/幼儿、可带长辈同行来设计，因此一期优先考虑低摩擦、可快速筛选、可二次确认的浏览体验。

## 核心场景

1. 今天或本周末在柏林有什么值得去的活动。
2. 1-3 小时内有什么适合带宝宝和长辈的短途活动、酒店、温泉、度假村。
3. 从柏林或德国机场出发，有哪些机酒套餐、全包酒店、阳光海岛、土耳其/埃及等目的地促销。
4. 有哪些邮轮、固定酒店多日游、Travelcircus/MSC/TUI/l'tur 类优惠值得关注。
5. 信息源很多时，可以按区域、类型、家庭友好度、来源和关键词快速过滤。

## 一期范围

- 接入公开 RSS 和可读 HTML 信息源。
- 每次数据更新时归一化标题、摘要、图片、价格、区域、类型、家庭友好评分和柏林相关性。
- 静态站点部署到 GitHub Pages，GitHub Actions 定时更新数据并重新部署。
- 文档化信息源目录、候选源、扩展路径和抓取策略。

## 暂不做

- 自动下单、自动登录、绕过反爬或付费墙。
- 保存家庭隐私数据、护照信息、账户凭据。
- 对价格做真实性保证。所有价格和库存都以源站点击后的结果为准。
- 完整旅行行程生成器。当前只提供路线规划框架和筛选入口。

## 评分假设

- 家庭友好：儿童、宝宝床、儿童泳池、Miniclub、厨房、短途、交通接送、全包等加分。
- 风险信号：Adults-only、夜生活为主、长途飞行、活动限制等减分。
- 柏林相关：柏林市内活动、Berlin/BER 出发、Brandenburg/Potsdam/Spreewald/Ostsee 等周边词加权。
